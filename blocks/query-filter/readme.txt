A WordPress plugin for filtering query blocks with various criteria.

== Description ==

This plugin provides a set of blocks for filtering WordPress queries, including taxonomy filters, keyword search, and more.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/gregius-query-filter` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
