Display SVG icons from a curated library of 150+ Lucide icons.

== Description ==

Choose from a carefully curated selection of 150+ modern icons including:
- Interface icons (menu, search, settings, etc.)
- Navigation icons (arrows, chevrons, etc.)
- Media icons (play, pause, volume, etc.)
- Communication icons (mail, phone, message, etc.)
- Business icons (briefcase, calendar, clock, etc.)
- And many more categories

Customize icon size, color, stroke width, and rotation for perfect integration with your design.

== Installation ==

This block is part of the RIAPress theme and is automatically available.
