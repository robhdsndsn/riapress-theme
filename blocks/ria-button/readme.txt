Branded call-to-action button block for RIA website.

== Description ==

The RIA Button block provides a customizable call-to-action button with multiple style variants and size options. Perfect for creating engaging CTAs throughout your RIA website.

**Features:**
* 4 style variants: Primary, Secondary, Tertiary, Ghost
* 3 size options: Small, Medium, Large
* Optional icons (left or right position)
* Open in new tab option
* Full URL picker integration
* Responsive and accessible

**Style Variants:**
* **Primary:** Main brand button with solid background
* **Secondary:** Alternative style for secondary actions
* **Tertiary:** Outlined button for subtle CTAs
* **Ghost:** Transparent button for minimal design

== Installation ==

This block is part of the RIAPress theme and is automatically available when the theme is active.
