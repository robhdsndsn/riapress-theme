An enhanced image block with advanced features for the RIA design system.

== Description ==

RIA Image is a powerful image block that extends WordPress core image functionality with:

**Aspect Ratios:**
- Square (1:1)
- Landscape (4:3)
- Widescreen (16:9)
- Ultrawide (21:9)
- Portrait (3:4)
- Custom aspect ratios

**Styling Options:**
- Border radius control (0-50px)
- Multiple border styles
- Box shadow effects
- Object fit options (cover, contain, fill, none)

**Hover Effects:**
- Zoom In - Image scales up on hover
- Zoom Out - Image scales down on hover
- Fade - Image opacity changes on hover
- Lift - Image elevates with shadow on hover
- Grayscale to Color - Desaturated to full color on hover
- None - No hover effect

**Overlay Options:**
- Overlay color with opacity control
- Show on hover only option
- Blend modes support

**Advanced Features:**
- Alt text for accessibility
- Image caption support
- Link to URL or media file
- Open link in new tab option
- Loading strategy (eager/lazy)

== Installation ==

This block is part of the RIA design system and is automatically registered with the RIAPress theme.
