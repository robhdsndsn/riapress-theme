Visual separator and section divider with multiple style variants.

== Description ==

Create visual separation between content sections with various styles:
- Solid line (default)
- Dashed line
- Dotted line
- Double line
- Thick line

Includes width control (narrow, medium, full) and spacing options.

== Installation ==

This block is part of the RIAPress theme and is automatically available.
