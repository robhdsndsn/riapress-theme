Branded heading block with customizable levels and style variants for RIA website.

== Description ==

The RIA Heading block provides semantic headings (H1-H6) with multiple style variants for different use cases. Perfect for creating a consistent typographic hierarchy throughout your RIA website.

**Features:**
* 6 heading levels (H1-H6) for semantic HTML
* 4 style variants: Default, Display, Subtitle, Label
* Text alignment options (left, center, right)
* Responsive font sizing with clamp()
* Color support via theme.json
* Toolbar controls for quick access
* Rich text editing with bold/italic support

**Style Variants:**
* **Default:** Standard heading with appropriate sizing for each level
* **Display:** Extra-large heading for hero sections (up to 5rem)
* **Subtitle:** Lighter weight for secondary headings
* **Label:** Small, uppercase text for section labels

== Installation ==

This block is part of the RIAPress theme and is automatically available when the theme is active.
