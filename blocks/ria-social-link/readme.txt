Individual social media link button with icons.

== Description ==

Display social media links with icons for:
- Facebook
- Twitter (X)
- LinkedIn
- Instagram
- YouTube
- Email

Includes two display styles:
- Icon Only (default)
- Icon + Label

== Installation ==

This block is part of the RIAPress theme and is automatically available.
