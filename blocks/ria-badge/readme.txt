Small status or label badge indicator with multiple color variants.

== Description ==

Display badges/tags/labels with various semantic color variants:
- Neutral (default gray)
- Info (blue)
- Success (green)
- Warning (yellow)
- Error (red)

Includes three sizes: Small, Medium, Large

== Installation ==

This block is part of the RIAPress theme and is automatically available.
